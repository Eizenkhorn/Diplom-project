# Открытые вопросы

## Метки на карте — полный список подтипов

**Статус:** требует уточнения у заказчика (ВНИИЖТ)

**Контекст:** В MVP используется следующий список подтипов меток, составленный по стандартной
легенде режимных карт ВНИИЖТ:

| Ключ | Название |
|------|----------|
| `entry_signal` | Входной светофор |
| `exit_signal` | Выходной светофор |
| `route_signal` | Маршрутный светофор |
| `passing_signal` | Проходной светофор |
| `crossing_guarded` | Переезд |
| `neutral_insert` | Нейтральная вставка |
| `ktsm` | КТСМ |
| `uksps` | УКСПС |
| `bridge` | Мост |
| `brake_marker` | Тормозной ориентир |
| `station_axis` | Ось станции |

**Вопрос заказчику:** Уточнить полный и окончательный список меток, используемых в
режимных картах. Возможные дополнения: предупредительные светофоры, знаки ограничения
скорости, локомотивные депо, посты ЭЦ, путевые ящики и т.д.

**Влияние на реализацию:** список хранится в `backend/models/markup_types.py` — добавить
новый подтип можно в одном месте без изменения модели данных.
